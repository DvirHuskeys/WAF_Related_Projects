# Journey: AE Alex (GTM Radar)

## Trigger
- Receives a cold domain list before an outreach sprint.
- Needs to prioritize accounts and tailor opening lines.

## Desired proof
- Confirm detected WAF/CDN stack (confidence level).
- Receive pain hypothesis (config drift, downtime, attack mix).
- Get a scripted opener + suggested next question.

## Flow
1. Upload CSV to GTM Radar.
2. Review prioritized list sorted by priority index.
3. Click a domain card to view stack snapshot + scores.
4. Copy “Pitch Angle” snippet into CRM.
5. Log follow-up action (e.g., request threat rehearsal artifact).

## Metrics
- 100 domains enriched per day locally.
- >70% detection confidence on WAF/CDN.
- Each AE call sheet contains at least one contextual hook.
